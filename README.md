# MysteriousPreserve.com

M.P.O. Website & Simple Jekyll Theme

## Feature

+ Darkmode
+ PWA support
+ Live2d widget
+ Beautiful design
+ Comment system
+ Syntax highlighting
+ Built-in musicplayer
+ Built-in search engine
+ Mobile-device friendly
